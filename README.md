# ego-Velocity Estimator for SPOT
ROS2 based **leg kinematic based ego-velocity estimator for Boston Dynamics SPOT**. Included in **GaRLILEO** and originally developed for **Co-RaL**. If you are planning to use this code for a part of your own project, **please cite one of the project**. Links for each projects can be found from below.
